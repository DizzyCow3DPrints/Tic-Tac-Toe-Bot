# Tic Tac Toe Bot
The Tic Tac Toe bot is a game of a tic tac toe that you play in the python output



```
Part of a real game:

Bot's Move: 
 X |   | O 
--- --- ---
   | X |   
--- --- ---
   | O |   
Enter your spot: 
```

# Features
1. A bot that uses random numbers to pick a spot, and it is pretty easy to beat
2. Uses python f strings to mix varibles and text to make the grid
3. Assigns the number 3 (for player) and 5 (for bot) for scoring since any combination of 3 numbers will not equal the 9 or 15 that got added for a 3 in a row to win
4. Coded to tell when there are draws
5. Has ditection on when you choose a already taken spot and has you try again
